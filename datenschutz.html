2. Zugriff auf die Website

Beim Aufruf dieser Website werden durch den technischen Betrieb des Webhostings möglicherweise automatisch Informationen erfasst. Dazu können insbesondere gehören:

  • IP-Adresse
  • Datum und Uhrzeit des Zugriffs
  • abgerufene Datei bzw. Seite
  • Browsertyp und Browserversion
  • Betriebssystem
  • Referrer-URL

Diese Verarbeitung erfolgt, um die Website technisch bereitzustellen und die Stabilität sowie Sicherheit des Betriebs zu gewährleisten.

3. Keine Cookies, kein Tracking, keine Analyse

Diese Website verwendet keine eigenen Cookies zur Analyse, kein Tracking und keine Werbedienste. Es werden keine personenbezogenen Daten von mir aktiv erhoben, gespeichert oder zu Marketingzwecken ausgewertet.

4. Externe Inhalte und technische Verbindungen

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Website kann es technisch erforderlich sein, dass dein Browser eine Verbindung zu Servern von GitHub bzw. dem technischen Infrastruktur-Anbieter hinter GitHub Pages aufbaut. Dabei können technisch notwendige Daten wie IP-Adresse, Datum und Uhrzeit, abgerufene Datei, übertragene Datenmenge und Browserinformationen verarbeitet werden.

Auf der Startseite werden außerdem öffentlich sichtbare Informationen aus GitHub-Repositories dynamisch geladen, um Projekte anzuzeigen. Dabei wird eine Verbindung zu GitHub hergestellt. Es werden von mir dabei keine eigenen Analyse- oder Tracking-Tools eingesetzt.
